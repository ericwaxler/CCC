Long Short-Term Memory

LSTM is a neural network architecture primarily used to predict time-series data.

Long Short-Term Memory is an artificial neural network architecture that can process sequences of data, in addition to single points. For this reason, they are particularly useful for analyzing time-series data where 'remembering' sequences of importance is critical.

Because sequence matters with time-series data, simple randomization is ineffective - the data must be split at a point in time.

There are two primary variables in this model:

  • Train/Test Split

  • The 'look-behind' period

The look-behind period defines the number of samples, in this case days, used to forecast the next data point. We'll examine several look-behind values using an 80/20 train/test split.

Look-Behind: 10 days

Look-Behind: 30 days

Look-Behind: 60 days

A 30 day look-behind period appears to produce the most accurate results. Next we'll manipulate the train/test split.
